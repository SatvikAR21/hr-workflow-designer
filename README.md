## üöÄ Overview  
The **HR Workflow Designer Module** is an interactive visual automation builder that enables HR teams to design and simulate workflows such as onboarding, approvals, and automated tasks.  

Built with **React**, **Vite**, and **React Flow**, this prototype demonstrates the ability to convert complex workflow logic into a clean, modular, and user-friendly interface with dynamic nodes, drag-and-drop behavior, and workflow simulation.

---

## üéØ Key Features  

### üîπ Interactive Workflow Canvas  
- Drag-and-drop custom nodes  
- Multi-directional connection handles  
- Smooth zoom, pan, and animated edges  
- Fully customizable node appearance  

### üîπ Custom Node System  
- Icons, titles, subtitles, and metric badges  
- Multiple node types (Start, Task, Approval, Validation, Trigger, End)  
- Designed for easy future extension  

### üîπ Mock API Integration  
- Fetches automation actions  
- Simulates workflow execution based on the graph  

### üîπ Workflow Simulation  
- Serializes workflow graph  
- Validates structure (missing nodes, cycles, invalid paths)  
- Shows step-by-step execution logs
